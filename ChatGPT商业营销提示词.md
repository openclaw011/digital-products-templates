## 一、电商运营类

### 1.1 电商产品描述生成
```
你是一位资深电商文案专家。请为以下产品生成吸引人的产品描述：

产品信息：
- 产品名称：{product_name}
- 核心卖点：{selling_points}
- 目标人群：{target_audience}
- 价格区间：{price_range}
- 风格要求：{style}

要求：
1. 开头抓住注意力
2. 突出产品核心价值
3. 解决客户痛点
4. 营造使用场景
5. 结尾引导购买

字数：{word_count}字
```

### 1.2 电商主图文案
```
生成电商主图文案：

产品：{product}
卖点：{main_benefit}

要求：
- 简短有力
- 适合主图展示
- 不超过10个字
- 引发点击欲望
```

### 1.3 电商详情页结构
```
生成电商详情页框架：

产品：{product}

需要包含：
1. 痛点场景引入
2. 产品介绍
3. 核心卖点（5个）
4. 使用场景
5. 用户评价
6. 品牌故事
7. 购买保障
8. 行动号召
```

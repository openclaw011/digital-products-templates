## 一、线上销售类

### 1.1 闲鱼无货源电商
```
项目原理：
在闲鱼平台卖货，从拼多多/1688进货，一件代发

操作步骤：
1. 注册闲鱼账号
2. 选品（选热门需求品）
3. 拼多多找同款
4. 加价上架闲鱼
5. 客户下单后拼多多下单
6. 赚取差价

优势：无需囤货、风险低、操作简单
月收入：1000-10000元
```

### 1.2 抖音小店无货源
```
项目原理：
开通抖音小店，搬运其他平台商品到抖音卖

1. 开通抖音小店
2. 选热门产品
3. 采集商品信息
4. 优化标题和主图
5. 挂车推广
6. 出单后采购发货

优势：流量大、门槛低
月收入：3000-30000元
```
